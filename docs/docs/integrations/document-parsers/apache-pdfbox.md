# Apache PDFBox

`ApachePdfBoxDocumentParser` can be found in the following module:
```xml
<dependency>
    <groupId>io.github.opensabe-tech</groupId>
    <artifactId>langchain4j-document-parser-apache-pdfbox</artifactId>
    <version>0.31.0.1</version>
</dependency>
```
